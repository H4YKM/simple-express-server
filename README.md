# Simple Express Server

<img style="width:100; height: 100;" src="https://upload.wikimedia.org/wikipedia/commons/thumb/d/d9/Node.js_logo.svg/1280px-Node.js_logo.svg.png" alt="Alt Text" title="Node Logo">

This is a simple site of three pages. The server of this site was created on Node.JS and on the express library.

First on terminal paste this code.

```javascript
npm install
```

After that run server in **dev** mode or in **final** mode.


**Dev** work in nodemon.
```javascript
npm run dev
```


**Start** work in default Node.
```javascript
npm run start
```
